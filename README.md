# dswTeXtools
Some basic LaTeX helpers. Really that's it!

## Setup
### Manual
I am using `pdflatex` and I place this `dswtextools.sty` into:
```
texmf/
└── tex/
    └── latex/
        └── dswtextools/
            └── dswtextools.sty*
```
Under my home directory.
### Automated
Use the included shell script. Run:
```console
./setup
```
